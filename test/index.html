CSS3 Color Tests

RGBa test

p.test {
  background: rgba( 0, 0, 0, .5 ) url( img/background.png ) top left repeat-x;
  border: 5px solid #999;
  color: rgba( 255, 255, 255, 1 );
  padding: 10px;
  }

This paragraph should have a 50% black background with a repeating image. The text should be 100% white.
